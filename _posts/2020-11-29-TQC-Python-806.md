# Python 806 字元次數計算
請撰寫一程式，讓使用者輸入一字串和一字元，並將此字串及字元作為參數傳遞給名為compute()的函式，此函式將回傳該字串中指定字元出現的次數，接著再輸出結果。

### 範例輸入1
```
Our country is beautiful
u
```
### 範例輸出1
```
u occurs 4 time(s)
```
