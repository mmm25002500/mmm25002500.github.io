# Python 310 迴圈公式計算
請使用迴圈敘述撰寫一程式，讓使用者輸入正整數n (1 < n)，計算以下公式的總和並顯示結果：
![image](https://latex.codecogs.com/gif.latex?%5Cfrac%7B1%7D%7B1&plus;%5Csqrt%7B2%7D%7D&plus;%5Cfrac%7B1%7D%7B%5Csqrt%7B2%7D&plus;%5Csqrt%7B3%7D%7D&plus;%5Cfrac%7B1%7D%7B%5Csqrt%7B3%7D&plus;%5Csqrt%7B4%7D%7D&plus;...&plus;%5Cfrac%7B1%7D%7B%5Csqrt%7Bn-1%7D&plus;%5Csqrt%7Bn%7D%7D)
> 輸出結果至小數點後四位。

### 範例輸入1
```
8
```
### 範例輸出1
```
1.8284
```
