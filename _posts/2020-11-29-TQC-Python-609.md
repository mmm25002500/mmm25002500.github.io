# Python 609 矩陣相加
請撰寫一程式，讓使用者建立兩個2*2的矩陣，其內容為從鍵盤輸入的整數，接著輸出這兩個矩陣的內容以及它們相加的結果。


### 範例輸入1
```
Enter matrix 1:
[1, 1]: 3
[1, 2]: 5
[2, 1]: 7
[2, 2]: 5
Enter matrix 2:
[1, 1]: 6
[1, 2]: 9
[2, 1]: 8
[2, 2]: 3
```
### 範例輸出1
```
Matrix 1:
**3 5 **
**7 5 **
Matrix 2:
**6 9 **
**8 3 **
Sum of 2 matrices:
**9 14 **
**15 8 **
```
