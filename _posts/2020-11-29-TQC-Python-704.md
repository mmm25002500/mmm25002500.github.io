# Python 704 集合條件判斷
請撰寫一程式，輸入數個整數並儲存至集合，以輸入-9999為結束點（集合中不包含-9999），最後顯示該集合的長度（Length）、最大值（Max）、最小值（Min）、總和（Sum）。

### 範例輸入1
```
34
-23
29
7
0
-1
-9999
```
### 範例輸出1
```
Length: 6
Max: 34
Min: -23
Sum: 46
```
