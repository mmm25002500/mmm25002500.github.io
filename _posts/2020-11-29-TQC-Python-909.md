# Python 909 聯絡人資料
請撰寫一程式，將使用者輸入的五個人的資料寫入data.dat檔，每一個人的資料為姓名和電話號碼，以空白分隔。再將檔案加以讀取並顯示檔案內容。

### 範例輸入1
```
Karen 123456789
Bonnie 235689147
Simon 987612345
Louis 675489321
Andy 019238475
```
### 範例輸出1
```
The content of "data.dat":
Karen 123456789

Bonnie 235689147

Simon 987612345

Louis 675489321

Andy 019238475
```
