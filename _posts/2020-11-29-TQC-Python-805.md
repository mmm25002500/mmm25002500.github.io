# Python 805 字串輸出
請撰寫一程式，要求使用者輸入一個長度為6的字串，將此字串分別置於10個欄位的寬度的左邊、中間和右邊，並顯示這三個結果，左右皆以直線 |（Vertical bar）作為邊界。

### 範例輸入1
```
python
```
### 範例輸出1
```
|python    |
|  python  |
|    python|
```
