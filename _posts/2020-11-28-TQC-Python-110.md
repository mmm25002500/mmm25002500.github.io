# Python 110 正n邊形面積計算
請撰寫一程式，讓使用者輸入兩個正數n、s，代表正n邊形之邊長為s，計算並輸出此正n邊形之面積（Area）。<br>
正n邊形面積公式: ![image](https://latex.codecogs.com/svg.latex?\frac{ns^2}{4tan(\pi/n)})
### 範例輸入
```
8
6
```
### 範例輸出
```
Area = 173.8234
```
