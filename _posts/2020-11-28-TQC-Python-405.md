# Python 405 不定數迴圈-分數等級
請撰寫一程式，以不定數迴圈的方式輸入一個正整數（代表分數），之後根據以下分數與GPA的對照表，印出其所對應的GPA。假設此不定數迴圈輸入-9999則會結束此迴圈。標準如下表所示：
|分數|GPA|
|---|---|
|90~100|A|
|80~89 |B|
|70~79 |C|
|60~69 |D|
|0~59  |E|
### 範例輸入1
```
75
```
### 範例輸出1
```
C
```
### 範例輸入2
```
39
```
### 範例輸出2
```
E
```
### 範例輸入3
```
100
```
### 範例輸出3
```
A
```
