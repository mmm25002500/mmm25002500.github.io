# Python 407 不定數迴圈-閏年判斷
(1) 請撰寫一程式，以不定數迴圈的方式讓使用者輸入西元年份，然後判斷它是否為閏年（leap year）或平年。其判斷規則如下：每四年一閏，每百年不閏，但每四百年也一閏。
(2) 假設此不定數迴圈輸入-9999則會結束此迴圈。

### 範例輸入1
```
2017
```
### 範例輸出1
```
2017 is not a leap year.
```
### 範例輸入2
```
2000
```
### 範例輸出2
```
2000 is a leap year.
```
### 範例輸入3
```
2016
```
### 範例輸出3
```
2016 is a leap year.
```
