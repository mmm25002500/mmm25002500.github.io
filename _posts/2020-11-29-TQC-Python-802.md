# Python 802 字元對應
請撰寫一程式，要求使用者輸入一字串，顯示該字串每個字元的索引。

### 範例輸入1
```
Kingdom
```
### 範例輸出1
```
ASCII code for 'K' is 75
ASCII code for 'i' is 105
ASCII code for 'n' is 110
ASCII code for 'g' is 103
ASCII code for 'd' is 100
ASCII code for 'o' is 111
ASCII code for 'm' is 109
713
```
