# Python 109 正五邊形面積計算
請撰寫一程式，讓使用者輸入一個正數s，代表正五邊形之邊長，計算並輸出此正五邊形之面積（Area）。<br>
正五邊形面積公式:![image](https://latex.codecogs.com/svg.latex?\frac{5s^2}{4tan(\pi/5)})

### 範例輸入
```
5
```
### 範例輸出
```
Area = 43.0119
```
