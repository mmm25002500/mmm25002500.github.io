# Python 908 單字次數計算
請撰寫一程式，要求使用者輸入檔名read.txt，以及檔案中某單字出現的次數。輸出符合次數的單字，並依單字的第一個字母大小排序。（單字的判斷以空白隔開即可）
[檔案下載](https://drive.google.com/uc?export=download&id=1pK-rWG7XiYBZzdxhyZO9q-awAd6KDIOR)

### 範例輸入1
```
read.txt
3
```
### 範例輸出1
```
a
is
programming
```
