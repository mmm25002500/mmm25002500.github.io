# Python 906 字串資料取代
請撰寫一程式，要求使用者輸入檔名data.txt、字串s1和字串s2。程式將檔案中的字串s1以s2取代之。
[檔案下載](https://drive.google.com/uc?export=download&id=1KWCiZYngdtIIJp3Y36QzrasTVa-0OLiI)

### 範例輸入1
```
data.txt
pen
sneakers
```
### 範例輸出1
```
=== Before the replacement
watch shoes skirt
pen trunks pants
=== After the replacement
watch shoes skirt
sneakers trunks pants
```
