# Python 403 倍數總和計算
請撰寫一程式，讓使用者輸入兩個正整數a、b（a<=b），輸出從a到b（包含a和b）之間4或9的倍數（一列輸出十個數字、欄寬為4、靠左對齊）以及倍數之個數、總和。

### 範例輸入1
```
5
55
```
### 範例輸出1
```
8   9   12  16  18  20  24  27  28  32  
36  40  44  45  48  52  54  
17
513
```
### 範例輸入2
```
4
9
```
### 範例輸出2
```
4   8   9   
3
21
```
