# Python 907 詳細資料顯示
請撰寫一程式，要求使用者輸入檔名read.txt，顯示該檔案的行數、單字數（簡單起見，單字以空白隔開即可，忽略其它標點符號）以及字元數（不含空白）。
[檔案下載](https://drive.google.com/uc?export=download&id=11KnroPBdIxOSzQSLJg_6hbx4ALg_xhJx)

### 範例輸入1
```
read.txt
```
### 範例輸出1
```
6 line(s)
102 word(s)
614 character(s)
```
