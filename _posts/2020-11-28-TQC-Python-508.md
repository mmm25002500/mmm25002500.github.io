# Python 508 最大公因數
請撰寫一程式，讓使用者輸入兩個正整數x、y，並將x與y傳遞給名為compute()的函式，此函式回傳x和y的最大公因數。>輸出的每一個字元後方有一空格。

### 範例輸入1
```
12,8
```
### 範例輸出1
```
4
```
### 範例輸入2
```
4,6
```
### 範例輸出2
```
2
```
