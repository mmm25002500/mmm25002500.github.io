# Python 810 最大值與最小值之差
請撰寫一程式，首先要求使用者輸入正整數k（1 <= k <= 100），代表有k筆測試資料。每一筆測試資料是一串數字，每個數字之間以一空白區隔，請找出此串列數字中最大值和最小值之間的差。
> 差值輸出到小數點後第二位。

### 範例輸入1
```
4
94 52.9 3.14 77 46
90.86(輸出)
-2 0 1000.34 -14.4 89 50
1014.74(輸出)
87.78 33333 29.3
33303.70(輸出)
9998 9996 9999
3.00(輸出)
```
