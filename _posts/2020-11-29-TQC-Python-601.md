# Python 510 費氏數列
請撰寫一程式，利用一維串列存放使用者輸入的12個正整數（範圍1~99）。顯示這些數字，接著將串列索引為偶數的數字相加並輸出結果。>費氏數列的某一項數字是其前兩項的和，而且第0項為0，第一項為1，表示方式如下：<br>
>輸出每一個數字欄寬設定為3，每3個一列，靠右對齊。

### 範例輸入1
```
56
45
43
22
3
1
39
20
93
18
44
83
```
### 範例輸出1
```
 56 45 43
 22  3  1
 39 20 93
 18 44 83
278
```
