# Python 308 迴圈位數加總
請使用選擇敘述撰寫一程式，讓使用者輸入三個邊長，檢查這三個邊長是否可以組成一個三角形。若可以，則輸出該三角形之周長；否則顯示【Invalid】。

### 範例輸入1
```
3
32412
```
### 範例輸出1
```
Sum of all digits of 32412 is 12
```
### 範例輸入2
```
0
```
### 範例輸出2
```
Sum of all digits of 0 is 0

```
### 範例輸入3
```
769
```
### 範例輸出3
```
Sum of all digits of 769 is 22
```
