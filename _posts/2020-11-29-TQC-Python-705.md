# Python 610 平均溫度
請撰寫一程式，依序輸入五個、三個、九個整數，並各自儲存到集合set1、set2、set3中。接著回答：set2是否為set1的子集合（subset）？set3是否為set1的超集合（superset）？

### 範例輸入1
```
Input to set1:
3
28
-2
7
39
Input to set2:
2
77
0
Input to set3:
3
28
12
99
39
7
-1
-2
65
```
### 範例輸出1
```
set2 is subset of set1: False
set3 is superset of set1: True
```
